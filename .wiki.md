# Project Summary
The AI Career Guidance Platform is a full-stack web application that provides AI-powered career guidance and job matching services for Rwandan youth. By leveraging advanced AI algorithms, the platform connects job seekers with suitable job opportunities based on their skills and preferences, while enabling employers to efficiently manage job postings and applications.

# Project Module Description
- **User Authentication**: Secure login and signup for both job seekers and employers.
- **Job Seeker Features**: 
  - Career profile management
  - Job browsing and searching
  - AI-suggested job matches
  - Job application tracking
- **Employer Features**: 
  - Job posting management
  - Application handling and candidate evaluation
- **AI Matching Component**: 
  - AI-based job matching algorithm for personalized job recommendations.

# File Description Inventory
- **README.md**: Overview and instructions for setting up the project.
- **.env**: Contains environment configuration variables.
- **package.json**: Lists project dependencies and scripts for Node.js.
- **server.js**: Initializes and configures the backend server.
- **routes/**: Contains all API route handlers for user, job, application, profile, and AI functionalities.
- **models/**: Defines the MongoDB schemas for various data entities.
- **public/**: Stores static files including uploaded documents.
- **client/**: Contains the React frontend application.

# Technology Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Frontend**: React.js
- **Authentication**: Session-based authentication
- **File Storage**: Local storage with Multer

# Usage
1. Install MongoDB locally or use MongoDB Atlas.
2. Create a `.env` file in the root directory with the necessary variables.
3. Install backend dependencies:
   ```
   npm install
   ```
4. Start the backend server:
   ```
   npm run dev
   ```
5. Set up the frontend:
   - Navigate to the client directory:
     ```
     cd client
     ```
   - Install frontend dependencies:
     ```
     npm install
     ```
   - Start the frontend:
     ```
     npm start
     ```
